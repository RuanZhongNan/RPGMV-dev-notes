# git diff 对比差异

- https://git-scm.com/docs/git-diff

## --summary 简要总结

```bash
git diff tovite conf-layout --summary
```

对比两个分支，输出格式为简单总结。

## --compact-summary

- https://git-scm.com/docs/git-diff#Documentation/git-diff.txt---compact-summary

```bash
git diff tovite conf-layout --compact-summary
```

对比两个分支，输出格式为紧凑总结。
