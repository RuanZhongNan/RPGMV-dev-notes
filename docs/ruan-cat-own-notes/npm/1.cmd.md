## 输出全局安装的依赖包 npm ls -g

- https://docs.npmjs.com/cli/v6/commands/npm-ls

指定为 json 格式输出

```bash
npm ls -g --json=true
```

查询指定深度，目前仅查询第一层即可。

```bash
npm ls -g --json=true --depth=0
```
