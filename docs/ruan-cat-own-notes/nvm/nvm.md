## 看清楚文档

- https://github.com/nvm-sh/nvm/blob/master/README.md#about
- https://github.com/coreybutler/nvm-windows

nvm-sh 不是用于 window 系统的，nvm-windows 才是。所以很多功能都没有。

## 下载最新版的指定 node 版本

下载最新版的 node14 ，特指为 lts 版本的。

```bash
nvm install 14 lts
```
