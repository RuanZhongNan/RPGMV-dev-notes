## 选购合适国外服务器供应商

选购便宜的国外服务器。不管选用的是哪个供应商提供的服务器，最核心的是获得：

- ip 地址
- 服务器账号
- 登陆密码

服务器系统一般选择 CentOS 7 64bit

## ssh 服务器链接的软件

MobaXterm 和 Xshell 都行。只要用上述提供的内容即可链接。

默认端口号 22
