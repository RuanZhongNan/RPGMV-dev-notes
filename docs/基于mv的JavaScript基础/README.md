这是*基于**RPGMakerMV**的**JavaScript**基础*，第二版作者：**阮中楠**

第二版相比于第一版的的内容，增加了以下内容：
- JavaScript ES6+ 语法的使用

第二版在语言措辞上会更加的简练，会详细的说明执行步骤。
