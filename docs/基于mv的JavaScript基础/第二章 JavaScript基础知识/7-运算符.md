# 7-运算符

## JavaScript 算术运算符
算术运算符用于执行变量与/或值之间的算术运算。给定```y=5```，下面的表格解释了这些算术运算符：

| 运算符 |       描述        | 例子  | 结果  |
| :----: | :---------------: | :---: | :---: |
|   +    |        加         | x=y+2 |  x=7  |
|   -    |        减         | x=y-2 |  x=3  |
|   *    |        乘         | x=y*2 | x=10  |
|   /    |        除         | x=y/2 | x=2.5 |
|   %    | 求余数 (保留整数) | x=y%2 |  x=1  |
|   ++   |       累加        | x=++y |  x=6  |
|   --   |       递减        | x=--y |  x=4  |

::: warning 不推荐使用繁杂的表达式来实现数据运算
第二版不推荐写类似于```x=++y```、```x=--y```的表达式来实现业务，这种写法可读性很差。不推荐读者在写插件代码时使用这样的写法。
:::
